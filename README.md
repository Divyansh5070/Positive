# Positive 🌟

An iOS app built with **SwiftUI** and **Firebase** to help users stay motivated, track their goals, and build consistent positive habits.

## Features

- 📅 **Calendar View** — Visualize your progress with a contribution heat map
- 🏡 **Home View** — Streaks, goal tracking, and daily highlights
- 🤝 **Buddy System** — Connect and grow with accountability partners
- 📝 **Feed** — Share and discover problem-solving entries with search & filters
- 🔐 **Authentication** — Secure login via Firebase Auth

## Tech Stack

- **SwiftUI** — Declarative UI framework
- **Firebase Firestore** — Real-time cloud database
- **Firebase Auth** — User authentication
- **Firebase Storage** — Media storage

## Setup

> ⚠️ This repo does **not** include the `GoogleService-Info.plist` for security reasons.

1. Clone the repo
2. Create your own Firebase project at [console.firebase.google.com](https://console.firebase.google.com)
3. Download your `GoogleService-Info.plist` and place it at:
   ```
   Positive/Positive/GoogleService-Info.plist
   ```
4. Open `Positive.xcodeproj` in Xcode
5. Build & run on your device or simulator

A template of the plist is provided as `GoogleService-Info.plist.example` for reference.
